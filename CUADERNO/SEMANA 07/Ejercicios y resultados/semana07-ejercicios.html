Ejercicios - Semana 07: Hooks en React

Inicio Cuaderno

Actividades Prácticas

Durante esta semana se desarrollaron ejercicios prácticos aplicando los principales Hooks de React en componentes funcionales.

useState

Se desarrolló un contador interactivo con botones para incrementar y decrementar el valor, utilizando el hook useState.

useEffect

Se implementó un reloj digital que actualiza cada segundo, gestionado mediante useEffect con un intervalo.

useContext

Se configuró un contexto para cambiar dinámicamente el tema de la aplicación (claro/oscuro).

Resultados Obtenidos

  • Entendí el flujo de estado con useState y cómo actualizarlo correctamente.
  • Aprendí a usar useEffect para ejecutar tareas después del renderizado.
  • Exploré useContext para evitar prop drilling y compartir datos globales.
  • Aumenté mi capacidad de crear componentes reutilizables con lógica encapsulada.
