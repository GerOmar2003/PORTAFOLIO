Semana 01 - Temas Aprendidos

Tecnologías de Desarrollo de Software

La evolución digital ha llevado al desarrollo de tecnologías orientadas al software moderno. Se destacan:

  • Tecnología Web: Basada en estándares abiertos, accesible desde cualquier navegador.
  • Software Nativo: Aplicaciones desarrolladas para un sistema operativo específico.

Open Web Platform

Conjunto de tecnologías abiertas desarrolladas por organizaciones como W3C, ECMA, etc. Incluye:

  • HTML5, CSS, JavaScript
  • SVG, MathML, WAI-ARIA
  • WebGL, APIs del navegador

Estándares Internacionales para el Desarrollo Web

Incluyen protocolos y tecnologías básicas y complementarias:

  • Básicas: HTTP/HTTPS, HTML, CSS, JavaScript, XML
  • Complementarias: APIs, SVG, WebGL, ARIA, MathML

Desarrollo Web

Frontend

HTML, CSS y JS. Lo que el usuario ve.

Backend

Python, PHP, Node.js. Lógica, base de datos, APIs.

UX/UI y Herramientas

  • UX: Experiencia del usuario, satisfacción
  • UI: Diseño de interfaces - Figma, Adobe XD
