🧠 Reflexión de la Semana 01

En esta primera semana, el proceso de familiarización con Visual Studio Code fue muy enriquecedor. Reconocer la interfaz de usuario, las funcionalidades de la paleta de comandos, el uso práctico de Git y GitHub desde la terminal integrada, así como la aplicación de Emmet y los atajos de teclado, me permitió agilizar considerablemente el flujo de trabajo en el desarrollo de código.

Además, el uso de herramientas como GitHub Codespaces refuerza la importancia de las tecnologías en la nube para la colaboración remota. Comprendí que dominar un entorno de desarrollo no solo mejora mi productividad, sino que también me acerca a prácticas profesionales del mundo real.

Esta práctica me deja como lección principal que aprender a usar eficientemente nuestras herramientas de trabajo nos hace desarrolladores más organizados, veloces y preparados para los retos que implica programar de forma profesional.
