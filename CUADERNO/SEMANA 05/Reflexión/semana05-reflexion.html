Reflexión - Semana 05: React

Inicio Cuaderno

¿Qué aprendí esta semana?

Durante esta semana aprendí a utilizar React, una biblioteca moderna y poderosa para el desarrollo de interfaces web. Descubrí cómo crear componentes funcionales, usar JSX para escribir estructuras HTML dentro de JavaScript y pasar información entre componentes usando props.

Me sorprendió lo intuitivo que puede ser trabajar con React cuando se comprende su lógica de componentes y el manejo del estado. También exploré herramientas como Create React App y Vite para crear entornos de desarrollo rápidos y eficientes.

Aprender React me motivó a pensar en la reutilización del código, en cómo organizar correctamente un proyecto, y en cómo separar lógica de presentación. Además, reforcé mis conocimientos sobre ES6, módulos, y sintaxis moderna de JavaScript.

Lo que aprendí

  • Componentes con funciones y props
  • Uso de JSX y estructura modular
  • Herramientas modernas como Vite y Webpack

Cómo me sentí

Me sentí motivado al ver cómo con pocas líneas de código podía tener una aplicación funcional. El poder de React me abrió los ojos a una nueva forma de pensar en la construcción de sitios web.

¿Qué mejoraré?

Seguiré practicando el uso de props, creación de componentes, y empezaré a trabajar con hooks para manejar estados. También quiero integrar React con otras herramientas como Firebase o Tailwind.
