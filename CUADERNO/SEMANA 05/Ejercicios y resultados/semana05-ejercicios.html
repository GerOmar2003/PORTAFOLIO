Prácticas Realizadas

En esta semana se desarrollaron ejercicios usando React, desde la creación del entorno hasta componentes funcionales. A continuación se muestran ejemplos representativos:

Componente Funcional

Se creó un componente funcional simple que imprime un mensaje en pantalla usando props.

JSX y Estilos

Se construyó una lista dinámica con datos mapeados desde un array de objetos.

Resultados Obtenidos

  • Crear e importar múltiples componentes.

  • Renderizar listas con map() y uso de key.
