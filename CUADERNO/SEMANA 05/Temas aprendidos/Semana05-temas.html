Semana 05: Desarrollo de Aplicaciones Web con React

Inicio Cuaderno

¿Qué es React?

React es una biblioteca de JavaScript desarrollada por Facebook para la creación de interfaces de usuario interactivas. Permite construir aplicaciones SPA (Single Page Application) reutilizando componentes y manejando eficientemente el estado de la aplicación.

Temas Principales

  • Instalación y uso de Node.js, Create React App y Vite.
  • Uso de JSX, componentes y props en React.
  • Herramientas como Babel, Webpack y ESLint.
  • Exploración de ReactDOM y React.createElement.
  • Aplicaciones modulares y estructura de carpetas en proyectos React.

Entorno React

React puede ser inicializado usando herramientas modernas como npx create-react-app o npm create vite@latest.

JSX y Componentes

JSX permite escribir HTML dentro de JavaScript, facilitando la creación de componentes dinámicos y reutilizables.

Herramientas modernas

Webpack, Babel y ESLint son clave para compilar, transpilar y analizar el código, asegurando rendimiento y calidad.

¿Qué aprendí?

Aprendí a estructurar una aplicación React desde cero, entender cómo funciona el flujo de componentes, manejar estados y props, y preparar el entorno con herramientas modernas. Además, comprendí cómo Babel transpila el JSX para que los navegadores lo comprendan, y cómo Webpack empaca todos los archivos en un bundle optimizado.
