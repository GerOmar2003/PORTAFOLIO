Ejercicios - Semana 06: JSX y TypeScript

Inicio Cuaderno

Prácticas Realizadas

Esta semana aplicamos nuestros conocimientos en React para crear componentes funcionales usando JSX y TypeScript. Se reforzó el uso de props tipadas, componentes reutilizables y estilos integrados.

Componente con props tipadas

Se creó un componente funcional en .tsx usando una interfaz para definir sus props y se renderizó con diferentes valores.

Condicionales en JSX

Se aplicaron expresiones ternarias para mostrar contenido dinámico dentro del componente según ciertas condiciones.

Estilos en React

Se usaron estilos inline y módulos CSS para personalizar los componentes, asegurando mantenibilidad y modularidad.

Resultados Obtenidos

  • Tipar correctamente los props de los componentes
  • Aplicar condiciones y bucles directamente en JSX
  • Integrar estilos con diferentes estrategias
  • Refactorizar código para mayor legibilidad
