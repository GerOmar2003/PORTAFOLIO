Reflexión - Semana 02

En esta segunda semana de clases, profundizamos en el entorno de desarrollo Visual Studio Code y comenzamos a aplicar herramientas más avanzadas como los selectores CSS, pseudo-clases, Emmet y los atajos de teclado.

Uno de los aspectos más destacables fue aprender a usar selectores específicos y pseudo-clases para estilizar elementos de forma precisa, lo cual nos permitirá diseñar interfaces más profesionales y funcionales. Además, con Emmet logramos escribir código HTML y CSS más rápido, aumentando nuestra productividad.

También reconocimos el valor de dominar los atajos del teclado, que aunque parecen simples, optimizan el tiempo al programar. A medida que nos familiarizamos con estas herramientas, nuestra confianza como desarrolladores va en aumento.

Lo que aprendí

  • CSS avanzado (selectores y pseudo-clases)
  • Prácticas con Emmet
  • Comandos rápidos y accesos de teclado

Cómo me sentí

Me sentí más cómodo y ágil utilizando Visual Studio Code. Las herramientas aprendidas me hacen sentir más preparado para crear proyectos reales.

¿Qué mejoraré?

Voy a seguir practicando los atajos y Emmet hasta que me resulten naturales, y buscaré crear mini proyectos para aplicar lo aprendido.
