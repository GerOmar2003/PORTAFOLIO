Semana 02: Visual Studio Code

Descarga e Instalación

Visual Studio Code está disponible en code.visualstudio.com, compatible con Windows, macOS y Linux.

¿Cómo utilizar VS Code?

  • UI - Pantalla Principal: Compuesta por barra lateral, panel principal, barra de estado, etc.
  • Paleta de Comandos: Con CTRL + SHIFT + P puedes acceder rápidamente a funciones como Create New Terminal o Emmet Remove Tag.
  • Terminal Integrada: Ejecuta comandos sin salir del entorno.
  • Emmet: Escribe HTML/CSS más rápido. Ver más en Emmet Docs.
  • Keyboard Shortcuts: Accesos rápidos para mayor eficiencia en la codificación.

Emmet

Emmet acelera el proceso de escribir código HTML y CSS mediante abreviaciones. Ejemplos:

  • ul>li*3 genera una lista con 3 elementos
  • div.container>h1{Hola Mundo}
