Semana 03: Dominando Emmet

¿Qué es Emmet?

Emmet es una herramienta poderosa que permite escribir código HTML y CSS de manera rápida mediante el uso de abreviaciones que se expanden automáticamente. Es ideal para desarrolladores front-end que buscan velocidad y eficiencia.

¿Por qué usar Emmet?

  • Ahorra tiempo al escribir estructuras repetitivas.
  • Reduce errores de sintaxis.
  • Mejora el flujo de trabajo al escribir HTML y CSS.
  • Compatible con VS Code, Sublime Text, Atom, entre otros.

Sintaxis común de Emmet

Elemento con clase

div.caja → <div class="caja"></div>

Elemento con ID

div#cabecera → <div id="cabecera"></div>

Multiplicación

ul>li*5 → 5 elementos <li> dentro de una <ul>

Atributos

input[type=text][placeholder=Nombre]

Combinaciones útiles

  • header+main+footer → Estructura básica de una página.
  • div.container>section.box*3 → Crea un contenedor con 3 secciones internas.
  • nav>ul>li.item$*4 → Añade clases numeradas automáticamente (item1, item2...)

Abreviaciones comunes

div.container

<div class="container"></div>

ul>li*3

<ul><li></li><li></li><li></li></ul>

a{Click aquí}

<a>Click aquí</a>

II. Fundamento Teórico

2.1. Plugin Emmet

Es una herramienta que utiliza operadores para agilizar tu proceso de desarrollo al escribir código HTML y CSS, más rápido y ahorrándote errores tipográficos.

OperadoresDescripción
>Hijo (dentro de...)
+Hermano (al lado de...)
^Salir del Tag actual
*Multiplicador (repite n veces)
()Agrupador
[attr]Añadir Atributos al Tag
{texto}Añadir contenido al Tag
$Añadir número de iteración
.Añadir class
#Añadir id

Etiquetas hermanos e hijos

section+section →
<section></section>
<section></section>

ul>li →
<ul>
  <li></li>
</ul>

Multiplicación de etiquetas

p*3 →
<p></p>
<p></p>
<p></p>

ul>li*3 →
<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>

Agrupar etiquetas

nav#menu>ul.nav-menu>(li.nav-menu-items>a)*3

Definir contenido

h1{Hola}

Clases

div.wrapper → <div class="wrapper"></div>
h1.header.center → <h1 class="header center"></h1>

ID’s

div#hero → <div id="hero"></div>

Atributos

img[src="cat.jpg"][alt="foto de gato"] → <img src="cat.jpg" alt="foto de gato" />

Enumerar iteración $

ul>li{Item $}*3 →
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

2.2. Atajos de Teclado de Visual Studio Code

General

  • Ctrl+Shift+P, F1 — Mostrar paleta de comandos
  • Ctrl+P — Buscar archivo
  • Ctrl+Shift+N — Nueva ventana
  • Ctrl+Shift+W — Cerrar ventana
  • Ctrl+, — Configuración
  • Ctrl+K Ctrl+S — Atajos de teclado

Edición Básica

  • Alt + flecha — Mover línea
  • Alt + Shift + flecha — Duplicar línea
  • Ctrl + Del — Borrar palabra
  • Ctrl + Shift + K — Borrar línea
  • Ctrl + Enter — Nueva línea abajo
  • Ctrl + Shift + Enter — Nueva línea arriba
  • Ctrl + L — Seleccionar palabra
  • Ctrl + Shift + L — Selección múltiple de palabra

Multicursor

  • Alt + click — Múltiples cursores
  • Ctrl + Shift + Alt + flecha — Múltiples cursores por teclado
  • Ctrl + D — Seleccionar palabra igual
  • Ctrl + U — Deseleccionar último
  • Shift + Alt + I — Insertar multicursor al final de líneas
  • Ctrl + Shift + L — Selección múltiple con cursores

Trucos avanzados de Emmet

  • {texto}: Permite insertar texto directamente. Ej: p{Hola mundo}
  • ^: Sube un nivel en la jerarquía del árbol HTML. Ej: div>ul>li^p
  • (): Agrupa elementos para mayor control. Ej: (header>nav)+main+footer
  • $: Inserta números secuenciales. Ej: ul>li.item$*3

Buenas prácticas

  • Combina Emmet con snippets y autoformato de tu editor.
  • No abuses de anidaciones profundas, mantén el código limpio.
  • Verifica el resultado expandido antes de guardar.
