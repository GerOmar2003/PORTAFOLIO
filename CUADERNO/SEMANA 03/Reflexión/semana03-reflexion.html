Reflexión - Semana 03

Durante esta tercera semana, nos adentramos a fondo en el uso del plugin Emmet, una herramienta esencial para acelerar el desarrollo web con HTML y CSS.

Descubrimos cómo utilizar operadores como >, +, *, {} y [] para crear estructuras complejas en segundos, así como cómo generar múltiples elementos dinámicos usando la sintaxis $.

También reforzamos nuestros conocimientos sobre atajos de teclado en Visual Studio Code, lo cual nos permite trabajar con mayor fluidez y eficiencia. El poder crear estructuras semánticas de forma automatizada es algo que sin duda potenciará nuestros proyectos web.

Lo que aprendí

  • Uso avanzado de Emmet
  • Operadores, agrupación, contenido, clases e IDs
  • Combinación de Emmet con atajos de VS Code

Cómo me sentí

Sentí que estoy comenzando a dominar herramientas reales de desarrollo profesional. Emmet me pareció sorprendente por su rapidez.

¿Qué mejoraré?

Practicaré más ejemplos con Emmet y buscaré aplicar sus funciones en mis propias páginas, mejorando así mi lógica al construir estructuras HTML.
