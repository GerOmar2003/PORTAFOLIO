HTML Avanzado

1. Introducción al HTML Avanzado

HTML avanzado incluye técnicas y etiquetas más especializadas que permiten crear estructuras más ricas, accesibles y semánticamente correctas.

  • Elementos semánticos avanzados
  • Multimedia
  • Integración con SVG y Canvas
  • Formularios enriquecidos

2. Elementos Semánticos Avanzados

HTML5 introdujo etiquetas que mejoran la accesibilidad y la estructura del contenido:

  • <section>, <article>, <aside>, <main>, <nav>, <header>, <footer>
<section>
  <article>
    <h2>Título</h2>
    <p>Contenido...</p>
  </article>
</section>

3. Multimedia

Permite insertar contenido de audio y video de forma nativa:

  • <audio> y <video> con atributos como controls, autoplay, loop.
<video controls width="300">
  <source src="video.mp4" type="video/mp4">
  Tu navegador no soporta video.
</video>

4. Gráficos SVG y Canvas

Permiten gráficos vectoriales (SVG) y animaciones o juegos (Canvas):

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="blue" stroke-width="4" fill="yellow" />
</svg>

5. Formularios Avanzados

Se utilizan nuevos tipos de campos de entrada y validaciones automáticas:

  • type="email", type="date", type="range"
  • Atributos: required, pattern, placeholder
<form>
  <label>Correo: <input type="email" required></label>
  <label>Edad: <input type="range" min="18" max="100"></label>
</form>

6. Atributos Personalizados (data-*)

Permiten agregar información personalizada a los elementos HTML:

<div data-usuario="omar123">Hola Omar</div>

7. Integración con JavaScript

HTML avanzado trabaja de la mano con JavaScript para crear interfaces dinámicas:

  • Eventos: onclick, onchange, etc.
  • Manipulación del DOM

8. Buenas Prácticas en HTML Avanzado

  • Usar etiquetas semánticas correctamente
  • Validar HTML
  • Evitar atributos obsoletos
  • Comentar adecuadamente

9. Ejercicio Final

Crea una página web que incluya:

  • Estructura semántica
  • Formulario con validaciones
  • Audio/video
  • Gráfico SVG
