Reflexión Personal

HTML Avanzado ha sido un descubrimiento poderoso. Ir más allá de las etiquetas básicas me ha abierto las puertas a la semántica web, accesibilidad, formularios dinámicos y a una estructura de contenido más profesional. Comprender el uso correcto de <section>, <article>, <aside>, e incluso atributos como aria-label o data-*, me permite escribir HTML más limpio, funcional y preparado para SEO.

Accesibilidad

Uso de atributos aria, etiquetas label bien asociadas, y roles que mejoran la experiencia para todos.

Data attributes

Permiten pasar información al frontend sin sobrecargar el HTML. Útiles con JS y librerías modernas.

Formularios completos

Con validación nativa, tipos de input variados, y mensajes personalizados.

¿Qué aprendí de HTML Avanzado?

Aprendí que escribir buen HTML no solo se trata de mostrar algo en pantalla, sino de pensar en el usuario, en los motores de búsqueda y en los desarrolladores que leen nuestro código. HTML avanzado me hace consciente de la estructura lógica y semántica que debe tener cada componente web.
