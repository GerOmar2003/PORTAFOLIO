¿Qué es CSS Avanzado?

CSS Avanzado nos permite construir interfaces modernas, interactivas y adaptables. Aquí exploramos técnicas como Flexbox, Grid, variables, animaciones, media queries y más.

Temas Avanzados de CSS

Flexbox

Permite distribuir elementos de forma eficiente en contenedores flexibles. Ideal para diseño responsive.

display: flex;
justify-content: center;
align-items: center;

CSS Grid

Una herramienta poderosa para crear diseños complejos de dos dimensiones.

display: grid;
grid-template-columns: repeat(3, 1fr);

Animaciones y Transiciones

CSS permite crear efectos visuales con `transition` y `@keyframes`.

transition: all 0.3s ease;
@keyframes mover {
  0% { transform: translateX(0); }
  100% { transform: translateX(100px); }
}

Variables CSS

Permiten reutilizar valores dentro de un archivo CSS, facilitando el mantenimiento del código.

:root {
  --color-principal: #1e40af;
}
h1 {
  color: var(--color-principal);
}

Media Queries

Ayudan a adaptar el diseño a distintos tamaños de pantalla.

@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

Pseudo-clases y Pseudo-elementos

Permiten aplicar estilos especiales a partes del documento o a eventos del usuario.

a:hover { color: red; }
p::first-letter { font-size: 2em; }
