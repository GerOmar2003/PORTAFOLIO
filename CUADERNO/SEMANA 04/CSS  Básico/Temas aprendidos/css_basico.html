¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que se utiliza para describir la presentación de un documento HTML. Gracias a CSS, podemos darle estilo, colores, fuentes, márgenes y mucho más a nuestras páginas web.

Temas Clave en CSS Básico

Selectores

Los selectores permiten seleccionar los elementos HTML que quieres estilizar. Ejemplos comunes son selectores de tipo, clase, ID y universales.

p { color: blue; }
#titulo { font-size: 24px; }
.container { padding: 10px; }

Colores

Se pueden aplicar usando nombres, valores HEX, RGB o HSL.

color: red;
color: #ff0000;
color: rgb(255, 0, 0);

Fuentes y texto

CSS permite modificar el tipo de letra, tamaño, alineación, espaciado y otros aspectos tipográficos.

font-family: Arial;
font-size: 16px;
text-align: center;

Bordes, márgenes y padding

Permiten controlar el espacio interno y externo de los elementos, así como sus bordes.

border: 1px solid black;
margin: 10px;
padding: 20px;

Display y posicionamiento

Define cómo se muestra un elemento: bloque, en línea, flex, grid, etc.

display: flex;
position: relative;
top: 10px;

Transiciones básicas

Permiten animar cambios en propiedades CSS.

transition: all 0.3s ease;
