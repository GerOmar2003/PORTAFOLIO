Reflexión Personal

Aprender CSS básico fue un gran paso para mí. Comprendí cómo darle estilo a un sitio web, haciendo que no solo funcione, sino que también se vea bien. Desde cambiar colores, tamaños de letra y márgenes, hasta usar fondos, bordes y animaciones simples, CSS me permitió expresarme visualmente en la web.

Colores y Fondos

CSS permite personalizar colores de texto, fondo y bordes para dar vida al contenido.

Texto y Tipografía

Aprendí a usar fuentes, tamaños, estilos y espaciado para mejorar la lectura.

Espaciado y Bordes

Con margin y padding ajustamos el espacio, y con border damos estructura visual.

¿Qué aprendí de CSS Básico?

Aprendí que CSS es más que solo embellecer. Es parte esencial de la experiencia del usuario. Dominar sus propiedades básicas me permitió tener control sobre el diseño y organización del contenido, lo que hace que una web sea clara, atractiva y profesional.
