¿Qué es Bootstrap?

Bootstrap es un framework de CSS desarrollado por Twitter que facilita el diseño web responsive con componentes ya listos y un sistema de rejilla.

Sistema de Grid

Bootstrap utiliza un sistema de 12 columnas que permite crear diseños adaptables. Puedes usar clases como .row y .col para organizar tu contenido.

Componentes de Bootstrap

  • Botones: btn btn-primary, btn btn-danger, etc.
  • Alertas: alert alert-success, alert alert-warning
  • Tarjetas: card para mostrar contenido agrupado
  • Formulario: clases como form-control, form-group
  • Navbar: navegación moderna con navbar

Utilidades

Bootstrap incluye clases utilitarias para márgenes (m-3), padding (p-2), colores de fondo, alineación, visibilidad, entre otros.

Responsive y Breakpoints

Bootstrap se adapta automáticamente a distintos tamaños de pantalla con clases como col-sm-, col-md-, col-lg-.
