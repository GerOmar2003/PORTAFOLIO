Reflexión Personal

Aprender Bootstrap me permitió comprender cómo agilizar el desarrollo de interfaces modernas, limpias y responsivas sin escribir grandes cantidades de CSS personalizado. Con componentes predefinidos, clases utilitarias y su sistema de grillas, puedo enfocarme más en la funcionalidad sin descuidar el diseño.

Sistema de Grillas

El grid system de Bootstrap facilita la creación de layouts responsivos con clases como .container, .row y .col.

Componentes

Permite usar botones, alertas, barras de navegación y otros elementos listos para usar.

Utilidades CSS

Clases como bg-primary, text-center y m-3 nos dan control rápido sobre el estilo.

Diseño Responsivo

Bootstrap se adapta automáticamente a dispositivos móviles con breakpoints inteligentes.

¿Qué aprendí?

Bootstrap me ayudó a diseñar interfaces más rápidas y adaptativas. Al dominarlo, puedo ahorrar tiempo sin sacrificar estética o usabilidad. Además, me motiva seguir explorando su integración con JavaScript para lograr experiencias más completas.
