HTML Básico

1. Introducción al HTML y estructura básica de un documento

1.1 ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Permite estructurar el contenido, como texto, imágenes, enlaces, formularios y más.

1.2 Estructura básica

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>

2. Etiquetas principales

Entre las etiquetas más utilizadas están: <h1>-<h6>, <p>, <a>, <img>, <br>, <strong>, <em>, etc.

2.1 Buenas Prácticas

  • Usar etiquetas semánticas correctamente
  • Evitar etiquetas obsoletas
  • Validar el código con W3C

3. Formato de Texto

  • <strong> para texto en negrita importante
  • <em> para énfasis (cursiva)
  • <br> para saltos de línea

4. Listas

HTML permite listas ordenadas (<ol>) y no ordenadas (<ul>).

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

5. Etiqueta Imagen

<img src="ruta.jpg" alt="Descripción de la imagen" />

5.1 Atributos

  • src: Fuente de la imagen
  • alt: Texto alternativo

5.2 Buenas prácticas

  • Optimizar imágenes para web
  • Siempre usar el atributo alt

6. Etiqueta Enlaces

<a href="https://ejemplo.com" target="_blank">Visitar</a>

6.1 Tipos de Enlaces

  • Internos (entre páginas del mismo sitio)
  • Externos (fuera del sitio)
  • Descargas o correos

6.2 Atributos

  • href
  • target
  • rel

6.3 Buenas prácticas

  • Usar enlaces descriptivos
  • Indicar si el enlace abre en una nueva pestaña

7. Tablas

7.1 Etiquetas básicas

<table>
  <tr><th>Nombre</th><th>Edad</th></tr>
  <tr><td>Ana</td><td>22</td></tr>
</table>

7.2 Atributos Principales

  • border, cellpadding, cellspacing

7.3 Etiquetas <thead>, <tbody> y <tfoot>

Estas permiten organizar la tabla en secciones lógicas.

7.4 Tablas estructuradas

Permiten mayor legibilidad y mantenimiento.

7.5 Tablas anidadas

Son tablas dentro de otras tablas (poco recomendadas).

7.6 Buenas prácticas

  • Evitar tablas para maquetado
  • Usar títulos claros

8. Formularios básicos

<form action="/enviar" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" />
</form>

8.1 Elementos básicos

  • <input>, <label>, <select>, <textarea>
