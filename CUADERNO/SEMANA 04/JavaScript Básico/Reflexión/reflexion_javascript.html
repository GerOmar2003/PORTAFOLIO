Reflexión Personal

Aprender JavaScript desde lo más básico me permitió entender cómo hacer mis páginas web interactivas. Desde mostrar alertas hasta modificar elementos HTML con eventos, este lenguaje es el alma de la web moderna.

Variables

Aprendí a declarar variables con let, const y var.

Operadores

Sumas, comparaciones, lógicos y concatenaciones me ayudaron a tomar decisiones.

Condicionales

El uso de if, else y switch me permitió manejar flujos lógicos.

Bucles

Los bucles for y while fueron clave para automatizar tareas repetitivas.

¿Qué aprendí?

JavaScript me demostró que no basta con hacer que un sitio se vea bien, también debe responder al usuario. Entender cómo funcionan los eventos, la manipulación del DOM y las funciones básicas me dio poder para hacer sitios más dinámicos. ¡Ahora sé que puedo crear desde una calculadora hasta un juego web!
