¿Qué es Tailwind CSS?

Tailwind es un framework de CSS de utilidad primero que permite crear interfaces modernas rápidamente mediante clases directamente en HTML.

¿Cómo se instala?

Puedes usar Tailwind directamente desde CDN para prototipos o instalarlo con NPM para proyectos personalizados:

  • Desde CDN: <link href="https://cdn.tailwindcss.com" rel="stylesheet">
  • Instalación: npm install -D tailwindcss y configuración con PostCSS

Filosofía Utility-First

Tailwind utiliza clases pequeñas y específicas como bg-blue-500, text-xl o p-4, que permiten construir cualquier diseño sin necesidad de escribir CSS personalizado.

Componentes personalizados

Aunque Tailwind no trae componentes predefinidos como Bootstrap, puedes crear los tuyos fácilmente o usar bibliotecas como Flowbite, DaisyUI o Headless UI.

Ventajas de usar Tailwind

  • Alta personalización sin escribir CSS adicional
  • Diseño responsivo con clases específicas
  • Rendimiento optimizado con PurgeCSS
  • Ideal para diseños a medida
