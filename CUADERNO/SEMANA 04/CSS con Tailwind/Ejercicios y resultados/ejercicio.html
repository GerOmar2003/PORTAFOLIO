Ejercicio y Resultados

Ejercicios y resultados prácticos usando HTML, CSS y JavaScript puro.

Instrucciones Generales

Esta práctica deberá desarrollarse sin librerías externas. Deberás trabajar con tu equipo en un repositorio de GitHub. Se califica la correcta distribución de tareas mediante ramas.

Ver repositorio en GitHub

Parte 1: Ruleta Dinámica

  • Texto dinámico desde TextArea
  • Control por botones y teclado (SPACE, S, E, R, F)
  • Animación de giro, selección aleatoria y ocultamiento

Parte 2: Sorteo de Equipos

  • Participantes editables en textarea
  • Definición por cantidad de equipos o miembros
  • Visualización aleatoria y organizada en tarjetas
  • Botones de descarga, copiar portapapeles y columnas
